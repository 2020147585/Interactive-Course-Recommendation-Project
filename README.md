# Interactive-Course-Recommendation-Project
📌 Overview (개요)
This project aims to solve the complexity of credit management and course selection for Computer Science students at Yonsei University. We developed an interactive dashboard that visualizes complex curriculum requirements and provides personalized course recommendations.
(본 프로젝트는 연세대학교 컴퓨터과학과 학생들의 복잡한 학점 관리와 과목 선택 문제를 해결하기 위해 기획되었습니다. 복잡한 커리큘럼 이수 요건을 시각화하고 개인별 맞춤형 수강 추천을 제공하는 인터랙티브 대시보드를 개발했습니다.)

🛠 Tech Stack (기술 스택)
Data Processing: Python (Pandas, NumPy)

Visualization: D3.js

Environment: Jupyter Notebook / GitHub (Collaboration)

🔑 Key Features (주요 기능)
Credit Visualization: A breakdown of completed vs. remaining credits by category (Major, General Education, Electives).
(전공, 교양, 선택 등 카테고리별 이수 학점 및 잔여 학점 시각화)

Course Roadmaps: Interactive UI showing prerequisite tree structures for CS courses.
(선후수 과목 체계를 시각화한 대화형 로드맵 제공)

Recommendation Engine: Suggests optimal courses based on the user's current grade and career interests.
(학년 및 진로 관심사에 따른 최적의 수강 과목 추천)

👤 My Role & Contributions (나의 역할 및 기여)
As a key developer in this collaborative project, I was responsible for the full-cycle development from data acquisition to UI component implementation:

Data Engineering (데이터 크롤링 및 수집):
Developed a web crawler to collect comprehensive prerequisite and course data across all years, structuring it into a CSV-based database for the system.
(모든 과목 및 연도별 선수과목 데이터를 수집하는 크롤러를 개발하여 시스템의 기초가 되는 CSV 데이터베이스를 구축했습니다.)

UI/UX Design & Prototyping (설계 및 프로토타이핑):
Designed the Figma prototype for the mid-term presentation, defining the user interaction flow and visual identity of the system.
(중간 발표를 위한 Figma 프로토타입을 제작하여 시스템의 사용자 인터페이스와 상호작용 흐름을 설계했습니다.)

Core Component Development (핵심 컴포넌트 개발):
Data Visualization: Developed line charts to visualize Mileage Cut-lines (enrollment priority scores) by subject, year, and major, enabling students to make data-driven decisions.
Logic Implementation: Built a timetable component with an automated time-conflict check feature to ensure valid course combinations.
(과목/연도/전공별 마일리지 커트라인을 라인 차트로 시각화하고, 강의 시간 중복 체크가 가능한 시간표 컴포넌트를 구현했습니다.)

Problem Solving & Leadership (문제 제기 및 해결):
Actively identified and addressed critical logic issues during development, such as optimizing the mileage distribution algorithm.
(프로젝트 진행 중 "마일리지 분배 알고리즘"의 논리적 오류를 찾아내고 수정 방향을 제기하는 등 시스템의 완성도를 높였습니다.)
